Executive Contracts Search

Analysts use this tool to look up client contracts by name. Unfortunately the search term is spliced directly into the SQL string.

Objective

Extract confidential data from the contracts database using SQL injection.

  • The search uses LIKE '%{your_input}%' - inject a UNION SELECT.
  • Determine the exact column count first (try ORDER BY 1, ORDER BY 2, etc.).
  • Extract the flag from a separate vault table.
